다시 이 저장소를 복제하여 commit 하는 법

1. cd Documents
2. git config --global user.name / user.email 설정
3. git clone "주소" // 이때, 이미 git 파일이 있으면 안됨
4. git code . // vs코드를 염
5. 작업 후 git add .
6. git commit -m "수정내용"
7. git push origin master/main(branch 이름을 봐라)
8. 끝!
